# Hack Club Links

This is our open-source Linktree alternative - an awesome modded version of [trevortylerlee/treelink](https://github.com/trevortylerlee/treelink)!

It's super simple to add new links - stick a new record in [siteConfig.json](https://github.com/leowilkin/hackclub-links/blob/main/src/siteConfig.json)
